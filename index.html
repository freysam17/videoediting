Curriculum

7 Modules to Mastery

A structured learning path designed to take you from complete beginner to confident intermediate editor.

01

Getting Started — The Premiere Pro Workspace

Master the interface and set up your first project

Topics Covered

  • Opening Premiere Pro and creating a new project with correct settings
  • Understanding the interface: Timeline, Program Monitor, Source Monitor, Project Panel
  • Customizing and saving your workspace layout for efficiency
  • Importing various media types: video, audio, and images
  • Organizing media into bins for a clean project structure

Key Takeaways

  • Confidently navigate the Premiere Pro interface
  • Set up projects with professional settings
  • Organize assets efficiently using bins
Mini Task

Import a set of provided clips and organize them into labeled bins (B-Roll, Interviews, Music, Graphics)

45-60 minutes
Practice files included
Beginner friendly
02

The Timeline — Where the Magic Happens

Learn essential editing techniques and timeline mastery

Topics Covered

  • Adding clips to the timeline using various methods (drag, insert, overwrite)
  • Basic playback and essential navigation keyboard shortcuts
  • Trimming, splitting & deleting clips using Razor Tool and Ripple Delete
  • Rearranging and repositioning clips on the timeline
  • Understanding video and audio tracks and layering

Key Takeaways

  • Edit clips quickly with professional shortcuts
  • Build and structure a complete rough cut
  • Manage multiple video and audio layers
Mini Task

Assemble a 60-second rough cut from provided clips, trimming and arranging for story flow

60-75 minutes
Practice files included
Shortcut cheat sheet
03

Transitions & Basic Effects

Add polish with smooth transitions and creative effects

Topics Covered

  • Adding video transitions: Cross Dissolve, Dip to Black, Dip to White
  • Adding audio transitions: Constant Power, Exponential Fade
  • Adjusting transition duration and placement for perfect timing
  • Exploring the Effects Panel and searching for effects
  • Applying basic effects: Crop, Gaussian Blur, Speed/Duration changes

Key Takeaways

  • Create smooth, professional scene transitions
  • Apply and modify basic visual effects
  • Control playback speed for creative effects
Mini Task

Add smooth transitions between scenes in your rough cut and apply a slow-motion effect to one clip

45-60 minutes
Effects reference guide
Video tutorial
04

Working with Audio

Master sound design, mixing, and audio enhancement

Topics Covered

  • Adjusting audio levels directly on the timeline with keyframes
  • Unlinking audio from video for independent editing
  • Adding background music and sound effects to your project
  • Basic audio mixing: balancing dialogue with music and effects
  • Using the Essential Sound Panel to tag and enhance audio
  • Basic noise reduction techniques for cleaner audio

Key Takeaways

  • Create professional audio mixes
  • Clean up and enhance dialogue audio
  • Layer music and sound effects effectively
Mini Task

Add a music bed to your project and balance it with dialogue, ducking music during speech

60-75 minutes
Music samples included
Audio levels guide
05

Titles, Text & Graphics

Create stunning titles, lower thirds, and motion graphics

Topics Covered

  • Creating titles using the Essential Graphics Panel
  • Customizing fonts, colors, size, stroke, and shadow
  • Designing professional lower thirds for interviews
  • Using and modifying Motion Graphics templates (MOGRTs)
  • Basic keyframing for text animation: position, opacity, scale

Key Takeaways

  • Design custom title graphics from scratch
  • Create animated text with keyframes
  • Use professional motion graphics templates
Mini Task

Create an animated intro title and a lower third name graphic for your project

60-75 minutes
MOGRT templates included
Font recommendations
06

Color Correction & Color Grading

Transform your footage with professional color techniques

Topics Covered

  • Understanding the Lumetri Color Panel and its sections
  • Basic correction: Exposure, White Balance, Contrast, Highlights, Shadows
  • Using Scopes (Waveform, Vectorscope, Parade) for accurate correction
  • Creative grading: Look, HSL Secondary, Vignette, Faded Film
  • Applying, customizing, and saving LUTs and presets
  • Matching color between different clips and cameras

Key Takeaways

  • Fix exposure and white balance issues
  • Create cinematic color grades
  • Match colors across multi-camera shoots
Mini Task

Color correct all clips in your project for consistency, then apply a creative cinematic grade

75-90 minutes
Free LUTs pack included
Scopes reference guide
07

Exporting & Final Workflow

Deliver your masterpiece in perfect quality for any platform

Topics Covered

  • Final review checklist before exporting your project
  • Understanding export settings: Format, Codec, Resolution, Frame Rate, Bitrate
  • Exporting directly vs. using Adobe Media Encoder for batch processing
  • Best export presets for YouTube, Instagram, TikTok, and general use
  • Creating and using proxies for smoother editing workflow
  • Organizing, backing up, and archiving completed projects

Key Takeaways

  • Export for any platform with optimal settings
  • Maintain quality while minimizing file size
  • Establish professional archiving habits
Mini Task

Export your final project in two formats: YouTube (1080p/4K) and Instagram Stories (vertical)

45-60 minutes
